# acracers
My own little attempt for creating a fixed-setup assetto corsa server with VERY fast track rotation.

This is pretty much finished. Implemented as a Discord bot running on the "Asseto Friends" server.

Next little subproject is a "roulette".
The idea is to have different cars in a race, and the server decides who drives which car depending on their skill.

So fast drivers will drive slow cars, slow drivers are in fast cars.

Tracks are mostly Kunos tracks.
Reference times are taken from RSRLiveTiming by radiators-champ.com.
The server can choose a car for a user for an upcoming race if the user participated in the previous
race. Their "skill" is being determined by their fastest lap from quali/race combined.
